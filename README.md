IPL Data Analysis Dashboard (2008–2025)
Overview

This project is an interactive data visualization dashboard built to analyze historical performance data of the Indian Premier League from 2008 to 2025.

It transforms raw cricket datasets into structured insights around teams, players, and season-level trends using interactive visuals, KPIs, and filters.

The focus is not decoration—it’s clarity, comparison, and decision-level insight extraction from sports data.

Key Features
Season Summary (KPI Layer)
Total matches played per season
Total teams and venues
Total boundaries (4s and 6s)
Centuries and half-centuries tracking
Season Highlights
Champion vs Runner-up visualization
Team logos and final match summary
Season-wise comparison views
Player Performance Analysis
Orange Cap: Highest run scorer of the season
Purple Cap: Highest wicket taker of the season
Top boundary hitters (4s and 6s leaders)
Points Table Analysis
Matches played, wins, losses, ties, no result
Points ranking system
Season-based filtering for trend analysis
